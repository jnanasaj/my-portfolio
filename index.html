Web Copilot

WebCopilot is an automation tool designed for comprehensive subdomain enumeration and bug detection using various open-source tools. It starts by enumerating subdomains with assetfinder, sublister, subfinder, amass, findomain, hackertarget, riddler, and crt.sh, followed by active enumeration using gobuster with SecLists wordlists. It verifies live subdomains using dnsx, extracts titles using httpx, and checks for subdomain takeover with subjack. Crawling endpoints with gau/gauplus, waybackurls, or waymore follows, filtering XSS, LFI, SSRF, SQLi, open redirect, and RCE parameters using gf patterns. Vulnerability scanning on subdomains employs tools like kxss, dalfox, openredirex, nuclei, etc. Results are printed and all outputs are saved in a specified directory.

All-in-One Malware Analysis

Malware analysis encompasses the examination of malicious software to uncover its behavior, capabilities, and potential impact. This involves static techniques like disassembly and string analysis to study code structure and metadata without execution. Tools such as IDA Pro and VirusTotal aid in this phase. Dynamic analysis, on the other hand, involves executing malware in controlled environments using tools like VMware and Cuckoo to observe real-time behavior and analyze network and system interactions with tools like Wireshark and Process Monitor.
